Master CSS

You must first understand the fundamentals of CSS to bring life and style to your web pages before diving into the storyline.

1. CSS Syntax

CSS is composed of rules. Each rule targets an HTML element and applies styles using properties and values.

selector {
    property: value;
}

/* Example */
h1 {
    color: blue;
    font-size: 24px;
}

Selector

Points to the HTML element you want to style (e.g., h1, p, div).

Property

The aspect of the element you want to change (e.g., color, margin, width).

Value

The specific setting you want to apply to the chosen property (e.g., blue, 20px).

2. Selectors

Selectors are how you target specific elements on the page.

/* Element Selector */
p { line-height: 1.5; }

/* Class Selector (starts with a dot) */
.highlight { background-color: yellow; }

/* ID Selector (starts with a hash) */
#main-header { text-align: center; }

Element

Targets all instances of a specific HTML tag across the entire page.

Class (.)

Targets elements that have a specific class attribute. Can be used on multiple elements.

ID (#)

Targets a single, unique element that has a specific id attribute.

3. Colors & Backgrounds

Bring your page to life by changing text and background colors.

.card {
    /* Text color */
    color: #ffffff; 
    
    /* Background color */
    background-color: #1a1629; 
    
    /* Background image */
    background-image: url('pattern.png');
}

Keywords

Simple predefined color names like red, blue, or transparent.

Hex Codes

A 6-character code starting with # representing RGB values (e.g., #ff0000 for red).

RGB / RGBA

Defines color using Red, Green, Blue values. RGBA adds an alpha channel for opacity.

4. The Box Model

Every HTML element is essentially a box. Understanding this is crucial for layout.

.box {
    width: 300px;
    padding: 20px;       /* Inside space */
    border: 2px solid;   /* The edge */
    margin: 15px;        /* Outside space */
}

Content

The actual content of the box, where text and images appear.

Padding

Clears an area around the content. Padding is inside the border.

Border & Margin

Border wraps the padding and content. Margin clears space outside the border to separate elements.

5. Typography

Control how your text looks with font properties.

body {
    font-family: 'Arial', sans-serif;
    font-size: 16px;
    font-weight: bold;
    text-align: center;
}

font-family

Specifies the font for an element. Always include a generic fallback family (like sans-serif).

font-size

Sets the size of the text. Can use pixels (px), ems (em), rems (rem), or percentages.

font-weight

Determines how thick or thin characters in text should be displayed (e.g., normal, bold).
