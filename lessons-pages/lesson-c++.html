Master C++

You must first understand the fundamentals of C++ before diving into the storyline.

1. The Basic Boilerplate

Every C++ program starts with a foundational structure. Here is the classic "Hello World".

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

#include <iostream>

A preprocessor directive that includes the standard input-output stream library, allowing you to print text.

int main()

The main function where program execution begins. It returns an integer (0) upon successful completion.

std::cout

The standard character output device (usually the screen). The << operator sends data to it.

2. Variables & Data Types

C++ is strongly typed, meaning you must declare the type of a variable before using it to store data.

int level = 5;               // Integer (whole numbers)
double health = 99.5;        // Double (floating-point numbers)
char rank = 'A';             // Character (single letter)
bool isGameOver = false;     // Boolean (true or false)

Numeric Types

Use int for standard whole numbers and double or float when you need decimal precision.

char

Stores a single character. It is always surrounded by single quotes (e.g., 'Z').

bool

Represents a boolean truth value. It can only evaluate to true (1) or false (0).

3. Control Flow

Direct the logic of your program using conditional statements and loops.

int score = 85;

// Conditional Statement
if (score >= 90) {
    std::cout << "Excellent!" << std::endl;
} else if (score >= 75) {
    std::cout << "Passed!" << std::endl;
} else {
    std::cout << "Keep trying!" << std::endl;
}

// Loop Statement
for (int i = 1; i <= 3; i++) {
    std::cout << "Spawning enemy " << i << std::endl;
}

if / else if / else

Allows your program to make decisions and execute different blocks of code based on whether a condition is true or false.

for loop

Used to repeat a specific block of code a known number of times. Contains an initialization, a condition, and an increment.

while loop

Repeats a block of code as long as a specified condition remains true. Useful when the number of iterations is unknown.

4. Functions

Reusable blocks of code designed to perform a specific task. They help keep your code organized and modular.

// Function Declaration & Definition
int addNumbers(int a, int b) {
    return a + b;
}

int main() {
    // Function Call
    int result = addNumbers(10, 5);
    std::cout << "The sum is: " << result;
    return 0;
}

Return Type

Specifies what type of data the function sends back (e.g., int, string). Use void if it returns nothing.

Parameters

Variables passed into the function (like int a, int b) so it has the data it needs to operate.

5. Arrays & Strings

Store multiple items of the same data type sequentially, or handle text efficiently using the string library.

#include <string>
using namespace std;

// Array of integers
int nums[5] = {10, 20, 30, 40, 50};
cout << "First number: " << nums[0] << endl; // Outputs 10

// String manipulation
string text = "C++ Programming";
cout << "Length: " << text.length();

Arrays

A fixed-size sequence of elements of the same type. Accessed using a zero-based index (e.g., nums[0]).

Strings

Use #include <string> to easily create and manipulate text. Includes helpful built-in functions like .length().

6. Pointers & References

Interact directly with memory addresses for dynamic allocation and passing variables efficiently.

int x = 10;

// Pointer
int *ptr = &x; 
cout << "Memory Address: " << ptr << endl;
cout << "Value: " << *ptr << endl;

// Pass by Reference Function
void changeValue(int &val) {
    val = 100;
}

Address Operator (&)

Retrieves the exact memory address of a variable in the computer's RAM.

Dereference Operator (*)

Used to access or modify the actual value stored at the memory address the pointer is holding.

References

Acts as an alias for an existing variable. Passing by reference allows functions to modify the original variable directly.

7. Structs

Structs allow you to group multiple related variables together into one custom data type. 

#include <iostream>
using namespace std;

// Define a struct
struct Player {

    string name;
    int hp;
    int level;

};

int main() {

    // Create a player object
    Player hero;

    hero.name = "Knight";
    hero.hp = 100;
    hero.level = 5;

    cout << hero.name << endl;
    cout << hero.hp << endl;
    cout << hero.level << endl;

    return 0;
}

Struct

A custom data type that groups related variables together.

Members

Variables inside a struct are called members.

Dot Operator

Use the dot operator to access struct members like hero.hp.
